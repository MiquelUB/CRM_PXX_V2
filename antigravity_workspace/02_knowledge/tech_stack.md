# Pila Tecnològica Completa
- **Base de Dades:** PostgreSQL allotjat a EasyPanel.
- **ORM:** SQLModel (Python).
- **Control de versions DB:** Alembic.
- **Client Web:** React.js.
- **Gestió de Peticions (Frontend):** SWR (Stale-While-Revalidate).
- **Desplegament:** EasyPanel sobre servidor Hetzner (Ubuntu).
