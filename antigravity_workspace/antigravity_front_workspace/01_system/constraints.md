# Stack i Límits
- **Framework:** React SPA (Vite).
- **Enrutament:** React Router. Estructura plana, sense nius innecessaris.
- **Estils:** Tailwind CSS. 
- **Mode Fosc:** Ús obligatori de l'estratègia `darkMode: 'class'` de Tailwind, guardant la preferència a `localStorage`.
- **Fetch & Memòria Cau:** Utilització exclusiva de SWR per revalidació.
