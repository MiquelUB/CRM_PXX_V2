# Arquitectura de Planes Definida
1. `/`: Dashboard amb Kanban i Calendari lateral.
2. `/deals/{id}`: L'Epicentre. Dades base + Agent IA (esquerra), Timeline i editor emails (dreta).
3. `/municipis`: Directori de poblacions (Llistat + CRUD).
4. `/contactes`: Directori de persones (Llistat + CRUD).
5. `/pagaments`: Gestor financer.
6. `/configuracio`: Administració.
